# On Call Prime Website

Premium corporate website for On Call Prime — an insurance-focused BPO company supporting carriers with claims operations, administrative workflows, customer support, and scalable back-office solutions.

---

## Overview

This project was built with a modern frontend stack focused on performance, premium UI/UX, smooth animations, and responsive design.

The website includes:

- Modern animated landing page
- Framer Motion page transitions
- Premium loading screen
- Cookie consent system
- Responsive layout
- SEO & Open Graph optimization
- GitHub Pages deployment
- Modern insurance/BPO branding

---

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion

## Installation

Clone the repository:

```bash
git clone https://github.com/kukavigan/on-call-prime.git
```

Navigate into the project:

```bash
cd on-call-prime
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

---

## Build

Create production build:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

---

## GitHub Pages Deployment

Install gh-pages:

```bash
npm install gh-pages --save-dev
```

Deploy website:

```bash
npm run deploy
```
